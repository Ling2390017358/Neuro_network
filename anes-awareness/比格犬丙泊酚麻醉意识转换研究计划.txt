# 丙泊酚诱导意识转换过程中比格犬丘脑-前额叶环路动态重构的 Neuropixels 研究计划

## 一、项目定位

### 1.1 推荐题目

**丙泊酚诱导意识丧失与恢复过程中，比格犬丘脑-前额叶环路的高密度电生理机制研究**

### 1.2 研究目标

本研究拟建立“比格犬 + 丙泊酚 + Neuropixels 高密度记录”的大型动物实验平台，连续记录丙泊酚诱导意识丧失（loss of consciousness, LOC）和意识恢复（recovery of consciousness, ROC）过程中的前额叶皮层（PFC）与丘脑活动，解析神经振荡、单神经元放电、spike-LFP 耦合和方向性信息流如何共同发生动态重构。

第一阶段目标应聚焦于一个可执行问题：

> 丙泊酚诱导 LOC/ROC 时，丘脑-前额叶环路是否从清醒或浅镇静基线下的灵活双向信息流，转变为以 alpha 频段同步锁定为主的低信息整合状态？

### 1.3 项目摘要

本项目以比格犬为大型动物模型，在个体化 MRI/CT 靶点规划指导下，利用 Neuropixels 高密度探针同步记录 PFC 与中央丘脑/高阶丘脑核团在丙泊酚诱导、维持和停药恢复全过程中的 AP 与 LFP 信号。研究以行为学 LOC/ROC 为时间锚点，结合视频、生理监护和给药记录，建立从宏观脑状态到单神经元放电模式的多尺度数据集。第一阶段不追求一次性覆盖所有意识相关网络，而是优先验证一个核心机制链条：PFC alpha 前化是否伴随丘脑 burst firing 增加、丘脑 spike 与 PFC alpha 相位锁定增强，以及 PFC-丘脑双向信息流下降。该结果将为麻醉意识转换的临界动力学、生物标志物提取和后续因果干预研究提供大型动物证据。

### 1.4 研究边界

第一阶段不建议同时纳入 DBS、多药物比较、TRN-听觉通路和全脑网络建模。上述内容可作为第二阶段扩展。本阶段优先完成：

1. 比格犬 Neuropixels 深部记录平台建立。
2. 丙泊酚诱导-维持-恢复流程标准化。
3. LOC/ROC 行为事件与电生理、生理监护、视频数据同步。
4. PFC-丘脑环路的主要电生理终点验证。

## 二、科学背景与核心假说

全身麻醉提供了一个可控、可逆地切换意识状态的实验窗口。清醒意识依赖丘脑与大脑皮层之间持续、动态、双向的信息交互。丘脑不仅是感觉信息中继站，也是皮层状态调控和信息门控的重要枢纽。前额叶皮层则参与高阶预测、反馈调制和跨区域信息整合。

丙泊酚主要增强 GABA_A 受体介导的抑制，可使丘脑-皮层环路从多频段、动态耦合状态转入低频同步状态。临床和基础研究均提示，丙泊酚麻醉过程中常出现前额叶 alpha 功率增强、皮层长程连接下降、丘脑神经元放电模式改变和信息整合能力下降。然而，现有头皮 EEG、fMRI 或低密度电极研究难以同时解析单神经元放电、LFP 振荡和方向性信息流。

### 2.1 核心科学问题

丙泊酚诱导 LOC 和 ROC 的临界过程中，PFC 与丘脑之间的神经振荡、单神经元放电和方向性信息流如何在毫秒至秒级时间尺度上发生转换？

### 2.2 核心假说

丙泊酚诱导意识丧失并非简单降低神经活动，而是使丘脑-前额叶环路从清醒或浅镇静状态下的多频段、灵活、双向交互状态，转变为以 alpha 频段同步化为主的低信息整合状态。该过程伴随丘脑神经元由 tonic firing 向 burst firing 转变、丘脑 spike 与 PFC alpha 振荡相位锁定增强，以及 PFC-丘脑方向性信息流下降。

### 2.3 工作模型

本研究采用“状态转换-环路重构-信息整合下降”的工作模型：

1. 丙泊酚增强 GABA_A 介导的抑制后，PFC 与丘脑进入更强的低频节律支配状态。
2. LOC 前后，PFC alpha 功率升高并可能出现前额叶优势，提示高阶皮层网络被窄带节律占据。
3. 丘脑神经元放电从 tonic firing 转向 burst firing，并更稳定地锁定在 PFC alpha 的特定相位。
4. 原本多频段、动态、双向的信息交换被 alpha 同步所替代，表现为 Granger causality、transfer entropy 或 PSI 等方向性指标下降。
5. ROC 不是 LOC 的简单倒放，恢复轨迹可能受药物残留、觉醒系统重新募集和网络惯性共同影响。

该模型的关键优势是可被证伪：如果 alpha 增强不先于 LOC、丘脑 burst 不增加、spike-LFP 锁相不增强，或方向性信息流不下降，则需要将 alpha 前化解释为伴随现象或下游结果，而不是意识转换的核心机制。

## 三、具体假设与可检验终点

### 3.1 假设 1：PFC alpha 功率升高先于或接近行为学 LOC

内容：丙泊酚诱导过程中，PFC LFP 的 alpha 频段（8-13 Hz）功率在 LOC 前或 LOC 附近显著升高，并强于感觉皮层对照区。

主要终点：

1. PFC alpha 功率相对于基线的变化幅度。
2. PFC alpha 功率变化起始时间与 LOC 时间的相对关系。
3. PFC 与 S1/V1 alpha 变化的空间差异。

预期结果：PFC alpha 功率在 LOC 前数十秒或 LOC 附近升高，S1/V1 变化较弱或滞后，支持 alpha 前化作为意识转换相关标志物。

### 3.2 假设 2：丘脑神经元放电由 tonic firing 转向 burst firing，并与 PFC alpha 相位锁定

内容：LOC 附近，中央丘脑或高阶丘脑核团神经元的放电模式由清醒/浅镇静基线下相对分散的 tonic firing，转变为更节律化的 burst firing，且 burst 或 spike 更集中地出现在 PFC alpha 振荡特定相位。

主要终点：

1. 丘脑神经元 burst rate。
2. burst 内 spike 数、burst duration、inter-burst interval。
3. 丘脑 spike 与 PFC alpha 相位的 PPC、PLV 或 spike-field coherence。
4. preferred phase 分布及其状态依赖性。

预期结果：LOC 后丘脑 burst firing 增加，spike-LFP 相位锁定增强，提示丘脑-前额叶环路被窄带 alpha 节律捕获。

### 3.3 假设 3：PFC-丘脑方向性信息流在 LOC 时下降，ROC 时恢复轨迹不完全对称

内容：清醒或浅镇静基线下，PFC 与丘脑之间存在双向动态信息流。丙泊酚诱导 LOC 后，PFC→丘脑反馈和丘脑→PFC 前馈方向的信息流均下降，并被窄带同步取代。ROC 时信息流恢复路径可能与 LOC 不完全对称，体现神经惯性。

主要终点：

1. PFC→丘脑与丘脑→PFC 的 Granger causality。
2. transfer entropy 或 phase slope index 作为交叉验证。
3. LOC 与 ROC 对齐后方向性指标的时间轨迹。
4. 前馈与反馈方向恢复时程差异。

预期结果：LOC 后双向信息流下降；ROC 时方向性连接恢复不完全沿 LOC 的反向路径进行。

### 3.4 主要终点、次要终点与探索性终点

为避免数据维度过高导致分析发散，第一阶段建议预先固定以下终点。

主要终点：

1. PFC alpha 功率在 LOC 对齐窗口内相对于基线的变化幅度和起始时间。
2. 丘脑神经元 burst rate 及 burst 结构在 LOC/ROC 前后的变化。
3. 丘脑 spike 与 PFC alpha 相位锁定强度，包括 PPC、PLV 或 spike-field coherence。
4. PFC→丘脑与丘脑→PFC 方向性信息流在 LOC/ROC 前后的时间轨迹。

次要终点：

1. PFC 与 S1/V1 的 alpha 功率变化差异。
2. gamma 功率、alpha/gamma 比值和跨频耦合变化。
3. PFC 浅层与深层神经元放电率和相位锁定差异。
4. LOC 与 ROC 的恢复路径是否存在滞后或不对称。

探索性终点：

1. 基于 LFP、spike 和生理指标组合预测 LOC/ROC 的模型。
2. 可区分“浅镇静-LOC-维持麻醉-ROC”的低维神经状态空间。
3. 与临床麻醉深度指标相衔接的 EEG/LFP 生物标志物。

### 3.5 建议固定分析时间窗

所有主要分析建议同时使用状态标签和事件对齐时间窗，避免仅凭药物剂量划分状态。

| 时间窗 | 建议定义 | 主要用途 |
|---|---|---|
| baseline | 给药前稳定 10-20 min | 个体内归一化 |
| early induction | 丙泊酚开始后至 LOC 前 120 s 之前 | 观察早期剂量效应 |
| pre-LOC | LOC 前 120 s 至 LOC 前 10 s | 识别前驱变化 |
| peri-LOC | LOC 前 10 s 至 LOC 后 30 s | 捕捉临界转换 |
| post-LOC | LOC 后 30-180 s | 稳定丧失意识后的环路状态 |
| maintenance | LOC 后稳定维持期 | 与临床麻醉状态对照 |
| pre-ROC | ROC 前 180 s 至 ROC 前 10 s | 识别恢复前驱变化 |
| peri-ROC | ROC 前 10 s 至 ROC 后 30 s | 捕捉恢复转换 |
| post-ROC | ROC 后 30-600 s | 观察恢复后的残余效应 |

## 四、实验总体设计

### 4.1 研究阶段

第一阶段采用急性或半急性记录策略，以平台建立和关键假设验证为目标。

建议样本：

1. 预实验：2 只比格犬。
2. 正式实验：6 只比格犬。
3. 每只正式动物尽量获得 2-3 次有效诱导-恢复 session，具体是否可重复取决于探针策略、动物福利评估和伦理审批。

### 4.2 动物选择

物种：健康成年比格犬，建议 2-5 岁，体重约 10-15 kg。

选择理由：

1. 比格犬具有沟回结构，脑体积和丘脑-皮层组织较啮齿类更接近人类。
2. 犬类 EEG 和麻醉相关频谱特征具有较好的临床转化价值。
3. 脑容量允许 Neuropixels NHP 长柄探针覆盖皮层和深部丘脑结构。
4. 可结合 MRI/CT、立体定位图谱和术后组织学进行靶点验证。

### 4.3 分组与状态

本阶段不设置多药物组，所有动物均接受丙泊酚诱导-恢复记录。

每个 session 至少包含以下状态：

1. baseline：清醒或浅镇静稳定基线。
2. induction：丙泊酚逐级诱导期。
3. pre-LOC：LOC 前时间窗。
4. post-LOC：LOC 后时间窗。
5. maintenance：稳定麻醉维持期。
6. pre-ROC：ROC 前时间窗。
7. post-ROC：ROC 后恢复期。

### 4.4 设计原则、对照与盲法

本研究以动物内重复测量为主要设计，重点比较同一动物、同一 session 内不同状态的动态变化，减少个体差异、头颅大小、探针路径和药代动力学差异对结论的影响。

关键对照包括：

1. 状态内对照：baseline、pre-LOC、post-LOC、maintenance、pre-ROC 和 post-ROC 之间比较。
2. 脑区对照：PFC 与 S1/V1 或其他感觉皮层对照区比较。
3. 方向对照：PFC→丘脑与丘脑→PFC 信息流分别估计。
4. 频段对照：alpha 与 delta、theta、beta、gamma 指标并列报告，避免只选择显著频段。
5. 生理对照：将 MAP、ETCO2、SpO2、体温和呼吸状态作为协变量或敏感性分析变量。

盲法建议：

1. LOC/ROC 视频标注由两名评估者独立完成，评估者不查看电生理结果。
2. spike sorting 质量复核时隐藏状态标签，仅保留通道、时间和单元质量信息。
3. 主要统计分析代码预先固定，先输出自动报告，再进行机制解释。

### 4.5 样本量与可行性依据

第一阶段 2 只预实验 + 6 只正式动物的设计更适合作为机制验证和平台建立研究，而不是一次性完成严格功效学确认。正式实验中每只动物争取 2-3 次有效 session，预计可形成 8-18 个高质量 session。主要统计单位仍为动物，session 作为重复测量进入混合效应模型。

预实验结束后应根据以下指标更新样本量判断：

1. PFC alpha 功率 LOC 前后变化的动物内效应量。
2. 丘脑高质量单元数量和跨 session 稳定性。
3. LOC/ROC 时间标注一致性。
4. 主要终点的缺失率和剔除率。

若预实验显示主要效应量稳定且缺失率低，可维持 6 只正式动物；若丘脑命中率或高质量单元产出不足，应优先增加预实验优化或增加动物数，而不是扩大分析问题。

## 五、记录脑区与设备配置

### 5.1 核心记录脑区

第一阶段必须优先保证两个核心区域：

| 脑区 | 目的 |
|---|---|
| 前额叶皮层 PFC | 检测 alpha 前化、层间活动和皮层输出变化 |
| 中央丘脑/高阶丘脑核团 | 检测丘脑门控、burst firing 和丘脑-皮层同步 |

条件允许时增加一个空间对照区：

| 脑区 | 目的 |
|---|---|
| S1 或 V1 | 判断 PFC alpha 增强是否具有前额叶优势 |

TRN 记录可作为第二阶段扩展，不作为第一阶段成功与否的必要条件。

### 5.2 探针配置

建议使用 Neuropixels 1.0 NHP 或其他经过大型动物深部记录验证的高密度探针。

第一阶段推荐配置：

1. 探针 A：PFC 至中央丘脑路径，作为核心探针。
2. 探针 B：S1 或 V1，对照探针。
3. 探针 C：仅在靶点规划、稳定性和安全性充分时考虑，用于 TRN 或额外丘脑核团。

如果预实验显示多探针增加出血、信号漂移或定位风险，应减少探针数量，优先保证 PFC-丘脑核心路径。

### 5.3 同步采集内容

必须同步记录：

1. AP 高频信号，采样率约 30 kHz，用于 spike sorting 和 burst 分析。
2. LFP 低频信号，采样率约 2.5 kHz，用于频谱、同步和信息流分析。
3. 视频。
4. 丙泊酚输注开始、剂量调整、停止输注时间戳。
5. 行为学 LOC/ROC 判定时间戳。
6. ECG、SpO2、ETCO2、血压、体温、呼吸频率。

所有关键事件应使用 TTL 或统一时间服务器写入采集系统，避免事后人工对齐造成误差。

### 5.4 丘脑靶区优先级

第一阶段不宜同时追求过多丘脑亚核团，应按问题优先级设定命中目标。

优先靶区：

1. 中央丘脑及板内核相关区域：更贴近觉醒调控和意识恢复问题。
2. 高阶丘脑核团：更贴近前额叶反馈调制和信息整合问题。

备选靶区：

1. 感觉丘脑核团：可作为前馈感觉通路对照，但不应替代核心 PFC-中央丘脑路径。
2. TRN：机制价值高，但手术和定位难度更大，建议作为第二阶段靶区。

通道定位应至少结合术前影像轨迹、术后影像/组织学和电生理特征三类证据。若三者不一致，应在主分析中采用保守脑区标注，并将细分核团分析降级为探索性结果。

## 六、术前准备与手术策略

### 6.1 术前影像与靶点规划

每只动物术前应完成 MRI 或 MRI/CT：

1. T1 高分辨率结构像，用于个体化脑图谱。
2. 必要时增加 DTI，用于避开重要白质束。
3. 将个体影像与比格犬脑图谱配准。
4. 规划探针入脑点、角度、深度和避血管路径。
5. 输出术中可执行的坐标表和轨迹图。

### 6.2 手术期麻醉与监护

手术期麻醉方案必须由兽医麻醉师制定并经伦理审批。原则上应明确区分：

1. 手术期麻醉：保证开颅、硬膜处理和探针植入安全。
2. 实验期麻醉：用于研究丙泊酚诱导 LOC/ROC 的神经机制。

如手术期使用异氟烷或其他麻醉药，应在正式丙泊酚记录前设置充分洗脱期，并记录呼气末浓度或其他可证明洗脱完成的指标，减少混杂。

### 6.3 第一阶段植入策略

建议第一阶段采用急性或半急性记录，不建议一开始开展完全慢性自由活动记录。

理由：

1. Neuropixels NHP 探针适合大型动物深部记录，但长期慢性植入需要额外工程保护。
2. 比格犬头颈运动、咀嚼和护理行为会增加探针损伤、感染和信号漂移风险。
3. 第一阶段核心目标是获得完整诱导-LOC-恢复过程的稳定记录，急性记录即可回答主要假设。
4. 平台成熟后，再设计半慢性微驱动或保护帽系统。

## 七、丙泊酚诱导与意识状态判定

### 7.1 实验流程

每个有效 session 建议包括：

1. 基线记录：20-30 分钟。
2. 丙泊酚阶梯式输注或靶控输注。
3. LOC 判定。
4. LOC 后维持记录：20-30 分钟。
5. 停止输注。
6. ROC 判定。
7. ROC 后恢复记录：30-60 分钟。

### 7.2 给药原则

丙泊酚给药参数应由兽医麻醉师根据动物体重、生命体征、机构 SOP 和伦理批件确定。研究方案中应预先写明：

1. 初始剂量或靶浓度。
2. 阶梯增加间隔。
3. 每一级维持时间。
4. 最大允许剂量或停止标准。
5. 低血压、呼吸抑制、低氧血症等情况的处理流程。

研究报告中必须完整记录实际输注速率、累计剂量、生命体征和任何干预。

### 7.3 LOC 判定

LOC 不能仅依赖药物剂量或 EEG，应结合行为学反应和视频记录。

可将 LOC 定义为连续两次或多次出现以下反应消失：

1. 对叫名或声音刺激无定向反应。
2. 对轻触耳廓、鼻端或足垫无定向反应。
3. 翻正反射消失或无法评估时记录替代反应。
4. 眼睑反射明显减弱或消失。
5. 下颌张力明显降低。
6. 无自主抗拒头固定或肢体牵拉。

具体判定标准应在预实验后固定，并在正式实验前写入 SOP。

### 7.4 ROC 判定

ROC 可定义为停药后重新出现以下至少两项：

1. 对声音刺激有定向反应。
2. 眼睑反射恢复。
3. 自主吞咽、眨眼或头部运动。
4. 对触觉刺激出现有目的撤回。
5. 肌张力恢复。

LOC 和 ROC 时间点均需由视频复核，最好由两名盲法评估者独立标注。

### 7.5 行为刺激流程标准化

LOC/ROC 判定本身会引入感觉刺激，因此需要在预实验后固定刺激强度、间隔和顺序。建议采用由弱到强的阶梯式流程：

1. 声音刺激：叫名或固定分贝短声，记录是否有定向眼动、耳动或头动。
2. 轻触刺激：耳廓、鼻端或肩部轻触，避免强痛刺激作为常规判定。
3. 体位或肌张力评估：在不影响探针稳定和动物安全的前提下进行。

刺激原则：

1. 每次刺激都写入 TTL 或事件表，不能只记录 LOC/ROC 总时间点。
2. LOC 判定期间刺激间隔保持一致，避免密集刺激改变觉醒状态。
3. 对无法完成某项反射评估的 session，应使用预先定义的替代指标。
4. 所有行为评分保留原始分项，而不是只保留最终 LOC/ROC 标签。

### 7.6 生理稳定性标准

为避免将低氧、低血压或高碳酸血症误解释为麻醉意识机制，正式实验应预先定义生理安全区间和分析剔除规则。至少应连续记录并在分析中报告：

1. 平均动脉压或无创血压。
2. SpO2。
3. ETCO2 或动脉血气替代指标。
4. 体温。
5. 呼吸频率、通气方式和任何机械通气参数。

若 LOC/ROC 关键窗口内出现持续严重低氧、循环不稳定或通气异常，该 session 应从主要机制分析中剔除，或仅作为安全性/探索性资料保留。

## 八、预实验计划

预实验不是为了验证机制结论，而是为了确认正式实验能稳定执行。

### 8.1 预实验目标

1. 验证 MRI/CT 靶点规划与实际入脑路径是否一致。
2. 验证探针能否稳定进入 PFC 和丘脑目标区。
3. 验证 AP 和 LFP 信号质量是否满足分析要求。
4. 验证丙泊酚诱导、LOC/ROC 判定和生命体征监护流程。
5. 验证 TTL、视频、输注泵和电生理数据同步。
6. 估算单次 session 原始数据量和处理时间。

### 8.2 进入正式实验的最低标准

预实验完成后，至少满足以下条件再进入正式动物记录：

1. 核心探针至少覆盖 PFC 和目标丘脑区域中的一个可信路径。
2. LFP 中可见稳定脑源性信号，且噪声通道比例可接受。
3. AP 信号可获得可排序 spike，至少部分通道存在高质量单元。
4. LOC/ROC 时间戳能与视频和电生理数据一致对齐。
5. 丙泊酚诱导过程中生命体征可控。
6. 数据命名、备份和预处理流程已经跑通。

### 8.3 预实验交付物

预实验结束后应形成一份内部技术报告，作为是否进入正式实验的决策依据。报告至少包括：

1. 每只动物的术前轨迹规划图、实际探针位置和偏差评估。
2. 每个 session 的 LOC/ROC 行为评分表和视频复核结果。
3. AP、LFP、视频、生理监护和给药记录的同步误差估计。
4. 通道级 QC 图，包括坏道比例、噪声水平、运动伪迹和 LFP 频谱。
5. spike sorting 初步结果，包括高质量单元数量和脑区分布。
6. 丙泊酚给药安全性总结和需要修改的兽医 SOP 条目。
7. 对正式实验样本量、探针数量和靶点优先级的更新建议。

## 九、数据分析计划

### 9.1 数据结构与命名

建议采用统一结构：

```text
Animal_ID/
  Session_ID/
    raw/
    events/
    physiology/
    video/
    preprocessing/
    spike_sorting/
    lfp_analysis/
    qc/
    figures/
```

每个 session 必须包含：

1. animal metadata。
2. session metadata。
3. 探针编号、入脑路径和目标脑区。
4. 给药记录。
5. LOC/ROC 标注。
6. 生命体征记录。
7. 数据质量控制表。

### 9.2 预处理

1. 统一电生理、视频、给药和生理监护时间轴。
2. 去除 50 Hz 及其谐波。
3. 标记运动伪迹、饱和通道和明显坏道。
4. 根据术后影像或组织学，将通道分配到 PFC 浅层、PFC 深层、白质和丘脑核团。
5. 为每个 session 生成 QC 报告。

### 9.3 LFP 分析

频段定义：

| 频段 | 范围 |
|---|---|
| delta | 1-4 Hz |
| theta | 4-8 Hz |
| alpha | 8-13 Hz |
| beta | 13-30 Hz |
| gamma | 30-100 Hz |
| high gamma | 100-200 Hz |

主要分析：

1. LOC/ROC 对齐的时频图。
2. PFC、丘脑和 S1/V1 的 alpha 功率时间曲线。
3. gamma 功率下降及 alpha/gamma 比值。
4. PFC-丘脑 coherence、PLV、wPLI 和 imaginary coherence。

### 9.4 单神经元分析

使用 Kilosort、Phy 或等效流程完成 spike sorting，并进行人工质控。

主要分析：

1. 不同状态下 firing rate。
2. 丘脑 burst rate、burst duration、burst 内 spike 数和 inter-burst interval。
3. PFC 浅层和深层神经元是否存在不同受影响时程。
4. LOC 敏感神经元亚群。

### 9.5 Spike-LFP 耦合

重点分析丘脑 spike 与 PFC LFP 的关系：

1. spike-triggered average。
2. spike-field coherence。
3. pairwise phase consistency。
4. preferred phase 分布。

核心判断：

> LOC 后丘脑神经元是否更集中地放电于 PFC alpha 振荡的特定相位？

### 9.6 信息流方向分析

采用多方法交叉验证：

1. Granger causality。
2. transfer entropy。
3. phase slope index。
4. 必要时使用 dynamic causal modeling 作为模型化扩展。

重点比较：

1. PFC→丘脑与丘脑→PFC 哪个方向先下降。
2. LOC 后是否由宽频双向交互转为窄带 alpha 同步。
3. ROC 时连接恢复是否与 LOC 轨迹对称。

### 9.7 状态空间与转换点分析

在完成主要终点分析后，可将 LFP 频谱、spike firing rate、burst 指标、spike-LFP 耦合和生理变量降维到统一状态空间，用于描述意识转换轨迹。

推荐分析：

1. PCA、UMAP 或 Gaussian process factor analysis 描述神经状态轨迹。
2. change-point detection 估计电生理转换点，并与行为 LOC/ROC 比较。
3. 隐马尔可夫模型或状态空间模型区分 baseline、induction、LOC、maintenance 和 ROC。
4. 在动物内训练 LOC/ROC 预测模型，在动物间做留一验证，避免只报告过拟合分类结果。

### 9.8 质量控制阈值

建议在正式分析前固定最低 QC 阈值，并在补充材料中公开每个 session 的 QC 表。

LFP QC：

1. 关键时间窗内有效通道比例达到预设阈值。
2. 50 Hz 及谐波处理后无明显残余线噪支配。
3. 运动伪迹时间段有明确标记，并从主要分析窗口剔除。
4. 相邻通道频谱和相位结构符合脑源性信号预期。

Spike QC：

1. 使用 isolation distance、L-ratio、ISI violation、amplitude drift 等指标筛选单元。
2. 对于 burst 分析，应排除明显受漂移或分选不稳定影响的单元。
3. 对状态依赖 firing rate 变化，应确认不是由探针漂移、噪声变化或分选阈值变化造成。

事件 QC：

1. TTL、视频帧时间、输注泵记录和电生理时间轴之间的同步误差需量化。
2. 若同步误差超过 peri-LOC/peri-ROC 窗口可接受范围，应降低该 session 的分析等级。
3. 人工修正事件时间必须记录修正原因和原始时间戳。

## 十、统计分析

### 10.1 统计单位

优先使用动物作为主要统计单位，session 作为重复测量。避免将大量通道或神经元简单当作完全独立样本。

### 10.2 推荐模型

使用线性混合效应模型：

```text
Y ~ State * Region + (1 | Animal) + (1 | Animal:Session)
```

其中 Y 可为 alpha 功率、PLV、burst rate、PPC、Granger causality 等指标。

### 10.3 多重比较与效应量

1. 使用 FDR 或 cluster-based permutation test 控制多重比较。
2. 报告效应量和 95% 置信区间。
3. 对 LOC 预测指标报告 ROC 曲线和 AUC。
4. 对主要终点进行预注册式固定，次要终点作为探索性分析。

### 10.4 时间序列统计与敏感性分析

事件对齐分析应避免逐点重复 t 检验。推荐策略：

1. 对时频图和连续时间曲线使用 cluster-based permutation test。
2. 对预先定义时间窗内的平均指标使用混合效应模型。
3. 对转换点时间差使用 bootstrap 置信区间或置换检验。
4. 对方向性指标同时报告原始方向值和方向差值，例如 GC(PFC→丘脑) - GC(丘脑→PFC)。

敏感性分析至少包括：

1. 排除生理波动较大的 session 后重复主要分析。
2. 使用不同 LOC/ROC 时间窗宽度重复分析。
3. 分别以动物均值和 session 均值为输入重复主要结论。
4. 将 MAP、ETCO2、SpO2、体温和丙泊酚累计剂量加入模型，检验结论是否稳健。

### 10.5 缺失数据与报告原则

1. 缺失原因按手术定位失败、信号质量不足、行为标注失败、生理不稳定和设备同步失败分类报告。
2. 不建议对关键电生理终点做简单插补；缺失 session 应在混合效应模型中按可用数据处理。
3. 所有剔除应先按预设 QC 标准执行，再查看主要结果，避免结果导向剔除。
4. 主文优先报告动物层面的效应，通道和单元层面的丰富数据放在补充图表或层级模型中呈现。

## 十一、成功标准与失败判定

### 11.1 单个 session 有效标准

一个 session 可纳入主要分析需满足：

1. LOC 和 ROC 至少一个事件时间点清晰。
2. PFC 或丘脑 LFP 信号稳定覆盖关键时间窗。
3. 关键生命体征无长时间严重异常。
4. TTL、视频和给药记录可对齐。
5. 运动伪迹未覆盖主要分析窗口。

### 11.2 动物层面有效标准

一只动物可纳入主要统计需满足：

1. 至少一个高质量 session。
2. 至少一个核心脑区定位可信。
3. 术后影像或组织学能支持通道定位。

### 11.3 剔除标准

以下情况应预先定义为剔除或仅作探索性分析：

1. 靶点严重偏离，无法确认 PFC 或丘脑信号来源。
2. 大量通道饱和或噪声过高。
3. LOC/ROC 判定无法复核。
4. 严重低氧、低血压或其他生理危机持续影响脑活动。
5. 关键时间戳缺失。

### 11.4 阶段性 go/no-go 标准

进入正式实验前：

1. 至少 1 只预实验动物获得可信 PFC 或丘脑 LFP，且事件同步完整。
2. 至少 1 个 session 能完成从 baseline 到 LOC 再到 ROC 的闭环记录。
3. 丙泊酚诱导流程未出现不可接受的安全风险。

完成前三只正式动物后：

1. 至少 2 只动物有可纳入主要分析的核心脑区记录。
2. PFC alpha、丘脑 burst 或 PFC-丘脑耦合中至少一个主要终点可被稳定估计。
3. 若丘脑命中率低于预期，应暂停扩大样本，先优化轨迹和固定流程。

第一阶段成功：

1. 至少 4 只正式动物进入动物层面主要统计。
2. 至少 6 个高质量 session 具有完整 LOC/ROC、PFC LFP 和同步记录。
3. 至少部分 session 获得可信丘脑单元，用于 burst 和 spike-LFP 耦合分析。
4. 能对三个核心假设给出支持、部分支持或否定性的明确结论。

## 十二、风险与替代方案

### 风险 1：丘脑靶点命中率不稳定

应对：

1. 强化术前 MRI/CT 个体化规划。
2. 使用立体定位图谱和术中电生理特征辅助判断。
3. 术后 MRI/CT 或组织学验证。
4. 预实验专门优化进针角度和深度。

### 风险 2：多探针增加出血或信号不稳定

应对：

1. 第一阶段优先单核心路径。
2. 必要时取消 TRN 或对照探针。
3. 采用慢速进针、硬膜微切口和充分止血。

### 风险 3：基线状态不是完全自然清醒

应对：

1. 第一阶段明确记录“清醒/浅镇静基线”的限制。
2. 使用最低可行镇静背景。
3. 重点分析同一 session 内的相对变化。
4. 第二阶段再发展半慢性自然清醒记录。

### 风险 4：数据量大、分析复杂

应对：

1. 预先固定 4 个主要终点。
2. 每个 session 生成自动 QC 报告。
3. 优先完成 LFP 和高质量单元分析。
4. 信息流分析只在通过 QC 的脑区和时间窗上进行。

### 风险 5：生理变量成为主要混杂因素

应对：

1. 与兽医麻醉师共同固定通气、补液和循环支持 SOP。
2. 将 MAP、SpO2、ETCO2 和体温纳入 QC 和统计模型。
3. 对生理异常 session 做敏感性分析。
4. 若某一生理变量与神经指标高度共变，应谨慎解释为麻醉意识机制。

### 风险 6：spike sorting 在麻醉状态转换期不稳定

应对：

1. 优先保证 LFP、MUA 和高质量单元三层结果均可报告。
2. 将明显漂移单元从单神经元主分析中剔除，但保留 MUA 作为辅助指标。
3. 对关键单元检查 waveform、amplitude 和 ISI violation 是否随状态系统性改变。
4. 对 burst 结论采用单元层面和动物层面双重汇总。

### 风险 7：行为 LOC/ROC 标准与犬类实验条件不完全匹配

应对：

1. 预实验中比较多种刺激和反射指标，选择最稳定且对探针干扰最小的组合。
2. 保留分项行为评分，避免单一二元标签过度简化。
3. 对 LOC/ROC 时间点使用评估者一致性检验。
4. 必要时将“意识转换”报告为行为反应消失/恢复，而不是过度推断主观意识内容。

## 十三、伦理、SOP 与启动前清单

实验启动前必须完成：

1. 动物伦理审批，包括 3R 原则、人道终点、镇痛、安乐死和术后护理。
2. 兽医麻醉 SOP，包括丙泊酚给药、气道管理、机械通气、循环支持和异常处理。
3. 手术 SOP，包括头架固定、开颅、硬膜处理、进针速度、止血和术后处理。
4. 数据同步 SOP，包括 TTL、视频、输注泵、生理监护和电生理系统。
5. LOC/ROC 行为判定 SOP。
6. 数据命名、备份、访问权限和质控 SOP。
7. 预实验通过标准。
8. 正式实验剔除标准。

### 13.1 动物福利与人道终点

伦理材料中应明确：

1. 动物来源、适应性饲养时间和术前健康筛查。
2. 术前禁食禁水、镇痛、抗感染和围手术期护理流程。
3. 探针植入、开颅和固定过程中的镇痛与麻醉深度监测。
4. 术后疼痛评分、神经功能观察、进食饮水和伤口护理记录。
5. 人道终点，包括持续无法控制的疼痛、严重感染、不可逆神经损伤、持续低氧/休克或其他兽医判定需终止实验的情况。
6. 安乐死方法和组织学取材流程。

### 13.2 3R 原则落实

替代：第一阶段仅回答大型动物必要问题，即跨皮层-深部丘脑的高密度同步记录与临床转化相关脑电特征，小鼠或离体系统不能完全替代。

减少：采用动物内重复 session、预实验 go/no-go 标准和分阶段推进，避免在技术流程未稳定时扩大样本。

优化：使用个体化影像规划、慢速进针、充分监护和明确终止标准，降低靶点失败、出血、感染和不必要痛苦。

---

## 十四、时间表

### 第 0 阶段：准备期，2-4 周

1. 整理伦理材料。
2. 完成麻醉、手术、行为判定和数据同步 SOP。
3. 建立 MRI/CT 靶点规划模板。
4. 搭建数据目录和分析脚本框架。
5. 完成设备联调和空跑测试。

### 第 1 阶段：平台建立与预实验，0-6 个月

1. 完成 2 只预实验。
2. 优化探针路径和植入流程。
3. 验证丙泊酚诱导-恢复流程安全性。
4. 跑通 LFP、spike sorting、事件同步和 QC 流程。

### 第 2 阶段：正式数据采集，6-18 个月

1. 完成 6 只正式动物记录。
2. 每只动物尽量获得 2-3 次有效 session。
3. 完成术后定位验证。
4. 建立标准化数据集。

### 第 3 阶段：数据分析，18-30 个月

1. 完成 LFP 频谱和同步分析。
2. 完成 spike sorting 和 burst firing 分析。
3. 完成 spike-LFP 耦合分析。
4. 完成方向性信息流分析。
5. 形成 LOC/ROC 动态模型。

### 第 4 阶段：论文与扩展，30-36 个月

1. 形成第一篇机制论文。
2. 提炼麻醉意识转换相关生物标志物。
3. 决定是否开展 DBS、多药物比较或半慢性自然清醒记录。

## 十五、预期结果与解释框架

### 15.1 若 PFC alpha 在 LOC 前升高

说明 alpha 前化可能是 LOC 的前驱神经标志，而非单纯的 LOC 后伴随现象。该结果可用于进一步构建 LOC 预测模型。

### 15.2 若丘脑 spike 与 PFC alpha 锁相增强

说明丘脑-前额叶环路可能被窄带 alpha 节律捕获，支持“动态门控失灵”假说。

### 15.3 若 PFC→丘脑反馈先下降

说明高阶皮层对丘脑门控的自上而下调控可能是意识丧失早期受损环节。

### 15.4 若 ROC 与 LOC 轨迹不对称

说明意识恢复不是麻醉诱导的简单反向过程，支持神经惯性或状态依赖性恢复机制。

若上述结果不成立，也应保留解释空间：alpha 前化可能是结果而非原因；丘脑放电改变可能是皮层同步化的下游表现；前馈与反馈可能平行下降而不存在明确先后顺序。

## 十六、创新点

1. **模型创新**：使用比格犬作为麻醉意识转换大型动物模型，弥合啮齿类与灵长类之间的转化缺口。
2. **技术创新**：在比格犬中建立 Neuropixels 深部高密度记录流程，同时覆盖皮层和丘脑。
3. **问题创新**：聚焦 LOC/ROC 临界转换期，而不是仅比较清醒与麻醉两个静态状态。
4. **分析创新**：在同一时间轴整合 LFP、单神经元、spike-LFP 耦合和方向性信息流。
5. **转化价值**：为临床麻醉深度监测和意识恢复研究提供更接近人类脑电特征的大型动物证据。

## 十七、数据管理与可重复性

### 17.1 数据分层保存

建议将数据分为四层：

1. 原始数据层：电生理原始文件、视频、输注泵记录、生理监护记录和原始事件文件，只读保存。
2. 预处理层：去噪、重采样、坏道标记、同步后的 LFP/AP 数据。
3. 分析层：spike sorting 输出、频谱矩阵、耦合指标、方向性指标和统计模型输入表。
4. 结果层：自动 QC 报告、统计表、图形、论文用数据和版本化分析脚本。

### 17.2 元数据标准

每个 session 至少记录：

1. 动物信息：编号、性别、年龄、体重、健康状态。
2. 手术信息：麻醉方案、开颅位置、探针型号、轨迹、入脑深度、进针速度。
3. 给药信息：丙泊酚浓度、输注速率、开始/调整/停止时间、累计剂量。
4. 行为信息：每次刺激时间、刺激类型、分项反应、LOC/ROC 判定者和复核结果。
5. 生理信息：血压、SpO2、ETCO2、体温、呼吸方式和异常处理。
6. 数据质量：坏道比例、噪声、运动伪迹、同步误差和剔除原因。

### 17.3 代码与分析复现

1. 建议为每个主要终点建立独立分析脚本和自动化报告模板。
2. 统计模型、剔除标准和主要图形生成代码应版本控制。
3. 每次分析输出应包含代码版本、参数文件和输入数据版本。
4. 关键结论应能从原始数据路径追溯到最终图表。

## 十九、最终总结

本研究第一阶段应以“丙泊酚诱导 LOC/ROC 过程中 PFC-丘脑环路动态重构”为唯一主线。最关键的成功标准不是一次性覆盖所有脑区和所有机制，而是稳定获得：

1. 可复核的 LOC/ROC 行为事件。
2. 可信的 PFC 与丘脑定位。
3. 高质量 LFP 和部分可排序单元。
4. 同步完整的视频、给药和生命体征数据。
5. 能回答四个主要终点的标准化分析流程。

完成第一阶段后，再根据结果决定是否扩展至 TRN、DBS、多药物比较或半慢性自然清醒记录。
