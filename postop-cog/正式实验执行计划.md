# 围术期神经认知障碍比格犬穿入式电极-Neuropixel-VR正式实验执行计划

版本：v1.1  
日期：2026-05-05  
适用文件：`麻醉后认知障碍神经生理机制研究课题申请书_修订版.md`

> 本计划用于把课题申请书转化为正式实验执行SOP。所有动物操作须在IACUC/动物伦理批准、兽医审核和机构生物安全/动物设施授权后开展；麻醉、镇痛、安乐死和意外处置以机构批准SOP为准。本计划不替代兽医处方或伦理批件。

---

## 1. 实验目标与核心假设

### 1.1 总目标

建立比格犬围术期神经认知障碍（PND/POCD）模型，利用听觉皮层/运动规划皮层穿入式电极、海马Neuropixel和“走/停”语音指令VR任务，同步刻画麻醉暴露与手术创伤对海马-皮层环路和空间行为的影响。

### 1.2 主要假设

1. 麻醉+手术组在术后急性期表现为“走”指令启动延迟、黑白马赛克奖励区停止准确性下降、“停”指令结束反应延迟和完整VR trial成功率下降。
2. 听觉皮层“走/停”语音诱发响应、运动规划皮层启动/奖励区停止/再启动/最终停止准备活动、海马位置细胞空间信息量和θ-γ耦合在术后急性期同步受损。
3. 多脑区神经指标的变化幅度与VR行为下降程度相关，且麻醉+手术组较单纯麻醉组更明显、恢复更慢。

## 2. 实验设计

### 2.1 分组

| 组别 | 干预 | 计划样本 | 主要目的 |
|---|---|---:|---|
| Control | 不接受正式实验性麻醉/手术，仅完成同等时间窗VR任务和神经记录 | n=4 | 重复测试和自然波动对照 |
| Anesthesia | 异氟烷维持麻醉60分钟，无手术创伤 | n=4 | 评估单纯麻醉效应 |
| Surgery | 异氟烷维持麻醉60分钟 + 标准化腹部小切口/肠管暴露操作 | n=4-5 | 评估麻醉叠加手术创伤效应 |

### 2.2 时间点

| 时间点 | 内容 | 关键输出 |
|---|---|---|
| 植入前 | MRI定位、健康评估、VR适应 | 个体化靶点规划、训练资格 |
| 植入后恢复期 | 穿入式电极 + Neuropixel植入、恢复、信号稳定性检查 | 可用通道数、单元产率、伤口恢复 |
| 行为基线期 | “走/停”语音指令VR训练和基线采集 | 完整trial成功率、奖励区停止成功率、基线神经信号 |
| T=0 | Control/Anesthesia/Surgery干预 | 麻醉/手术记录、生命体征、苏醒时间 |
| T=2h | 清醒后VR和神经记录 | 急性期行为和环路变化 |
| T=6h | VR和神经记录 | 早期恢复 |
| T=24h | VR和神经记录 | 次日恢复 |
| T=7d | VR和神经记录，必要时终末组织学 | 迟发效应和组织学验证 |

## 3. 启动前准备清单

### 3.1 必须完成

| 项目 | 负责人 | 完成标准 |
|---|---|---|
| IACUC/动物伦理批准 | PI | 批件覆盖电极植入、麻醉、手术、VR训练、终末取材 |
| 兽医SOP审核 | 兽医/麻醉负责人 | 麻醉、镇痛、抗感染、补液、苏醒监护、终止标准明确 |
| 手术团队培训 | 手术负责人 | 至少完成模拟定位和设备装配演练 |
| VR系统联调 | 行为负责人 | “走/停”语音、跑台、直线迷宫滚动、黑白马赛克奖励区、食物奖励、TTL、视频同步成功 |
| 数据采集联调 | 电生理负责人 | 穿入式电极和Neuropixel同步采集、时间戳一致 |
| 数据模板建立 | 数据负责人 | 建好文件命名、CRF、质控表、备份脚本 |

### 3.2 Go/No-Go标准

进入正式动物实验前，至少完成2次无动物全链路演练：

- 语音“走/停”TTL与神经采集时间戳偏差 < 1 ms。
- 跑台速度、VR位置、黑白马赛克奖励区出现、食物奖励、视频帧、IMU和神经数据可按同一trial ID合并。
- 单次40分钟任务无采集软件崩溃、无数据丢包。
- 数据当天可完成双备份和质控报告。

## 4. 动物入组与随机化

### 4.1 入组标准

- 健康成年比格犬，1.5-2.5岁，体重12-16 kg。
- 健康检查、基础血液学/生化、心肺评估和术前MRI无排除性异常。
- 能完成跑台适应，无明显恐惧、攻击或持续拒训。

### 4.2 排除标准

- 神经系统、心肺、严重感染或行为异常。
- MRI显示目标脑区结构异常。
- 训练期无法达到最低VR任务响应标准。
- 植入后出现持续疼痛、感染、神经功能异常或信号质量不可用，经兽医/PI判定不宜继续。

### 4.3 随机化

在完成VR基础适应后按区组随机分配至Control、Anesthesia、Surgery三组。分配前记录体重、性别、年龄、训练表现，尽量平衡组间基线。

## 5. 电极植入与恢复流程

### 5.1 靶点规划

| 脑区 | 电极 | 目的 | 定位方式 |
|---|---|---|---|
| 海马CA1-CA3-DG轴 | Neuropixel | 位置细胞、海马LFP、E/I、θ-γ PAC | 术前MRI + 犬脑图谱 + 术中LFP确认 |
| 听觉皮层 | 穿入式电极 | “走/停”语音诱发响应 | 术前MRI个体化规划 |
| 运动规划皮层 | 穿入式电极 | 步行启动/停止准备活动 | 术前MRI个体化规划 |

### 5.2 手术当日步骤

1. 术前确认：禁食/禁水按兽医SOP执行，核对动物ID、体重、批件编号、植入靶点图。
2. 麻醉与监护：由兽医或授权麻醉人员执行；记录诱导、维持、镇痛、补液、体温、SpO2、ETCO2、心率、血压。
3. 定位与植入：先完成穿入式电极植入，再完成Neuropixel定位和下降；记录每个靶点坐标、下降深度、阻抗/噪声和初始LFP特征。
4. 固定与封闭：固定驱动器和连接器，完成伤口封闭，拍照记录头部组件状态。
5. 苏醒监护：记录自主呼吸、吞咽/睫毛反射、站立、进食饮水和疼痛评分。

### 5.3 术后恢复

| 时间 | 操作 | Go/No-Go |
|---|---|---|
| 0-72h | ICU或强化监护，疼痛评分、体温、进食、伤口、神经状态 | 无不可控疼痛、无活动性出血/感染 |
| Day 4-7 | 轻度活动恢复，头部组件检查，短时神经信号测试 | 噪声不过高，动物能耐受连接 |
| Day 8-14 | 逐步恢复跑台和记录装置适应 | 行为稳定，无头部组件松动 |
| Day 14+ | 进入正式VR训练前评估 | 兽医和PI共同批准 |

## 6. “走/停”语音指令VR任务SOP

### 6.1 硬件与同步

- 语音来源：优先使用标准化录音；如使用训练者现场发声，需固定训练者、音量、语速和距离。
- 指令：仅使用“走”和“停”两个中文指令。
- 同步：每次语音起始同时发送TTL，记录到神经采集、VR日志、跑台控制和视频系统。
- 行为检测：跑台速度、IMU、红外追踪和视频共同判定步行启动/停止。
- 奖励系统：使用可定量投放的食物奖励装置；奖励投放、奖励获取和摄食完成时间均进入VR日志。
- VR场景：采用虚拟直线迷宫。比格犬开始步行后，跑步机传送带转动并带动屏幕中的虚拟直线迷宫向前滚动；场景中设置黑白马赛克标记地带作为奖励停止区。

### 6.2 单个trial定义

| 阶段 | 触发 | 成功判定 | 记录变量 |
|---|---|---|---|
| Trial开始 | 播放或发出“走” | 犬在预设时间内开始连续步行 | `go_ttl`、启动潜伏期、初始速度 |
| 直线迷宫滚动 | 检测到连续步行 | 跑步机传送带转动，VR直线迷宫开始向前滚动 | 跑台速度、VR位置、头动、视频 |
| 奖励区出现 | VR场景进入黑白马赛克标记地带 | 犬识别马赛克地带并主动停止走动 | `mosaic_onset_ttl`、到达速度、停止潜伏期 |
| 奖励获取 | 犬在黑白马赛克地带内停止并保持静止 | 投放食物奖励，犬完成摄食 | `reward_ttl`、`reward_acquired_ttl`、`consumption_done_ttl`、停止位置误差、摄食时长 |
| 继续前进 | 摄食完成后 | 犬重新开始向前走，直线迷宫继续滚动 | `resume_ttl`、再启动潜伏期、恢复速度 |
| Trial结束 | 播放或发出“停” | 犬在预设时间内停止走动并保持静止 | `stop_ttl`、结束停止潜伏期、最终停止位置、trial成功/失败 |

### 6.3 错误类型

- 启动失败：听到“走”后未在预设时间内开始步行。
- 奖励区提前停止：黑白马赛克地带出现前停止。
- 奖励区漏停：黑白马赛克地带出现后未停止或明显越过奖励区。
- 奖励获取失败：已在奖励区停止但未摄食、摄食中断或无法在限定时间内完成摄食。
- 继续前进失败：摄食完成后未重新开始向前走。
- 结束失败：听到“停”后未停止或停止潜伏期超出阈值。
- 迷茫反应：方向/速度异常、频繁停走、无法完成trial。

### 6.4 训练阶段

| 阶段 | 时间 | 目标 | 进入下一阶段标准 |
|---|---|---|---|
| 环境适应 | Day 1-3 | 适应跑台、头部连接和实验员 | 无明显恐惧，能站立/慢走 |
| 视觉VR适应 | Day 4-7 | 适应虚拟直线迷宫滚动和食物奖励 | 能完成短时跑道推进 |
| 语音关联训练 | Day 8-10 | “走”=开始步行并启动迷宫滚动，“停”=trial结束停止 | 指令响应率 > 40% |
| 马赛克奖励区训练 | Day 11-21 | 看到黑白马赛克地带后停止并获取食物奖励 | 奖励区停止成功率 > 60% |
| 稳定化训练 | Day 22-42 | 完成“走”启动-马赛克停止取食-继续前进-“停”结束的完整trial | 完整trial成功率 > 70%，连续3次session变异系数 < 10% |

## 7. 正式实验日流程

### 7.1 实验前一天

- 确认动物健康、进食、体重、伤口和头部组件。
- 运行10分钟设备测试：语音TTL、直线迷宫滚动、黑白马赛克奖励区、食物奖励装置、跑台、视频、神经采集。
- 建立当日文件夹：`YYYYMMDD_DogID_Group_Timepoint/`。
- 准备CRF、药品耗材、急救设备和备份硬盘。

### 7.2 T=0干预

| 组别 | 操作 | 必记变量 |
|---|---|---|
| Control | 同等时间窗安静固定/连接记录，不麻醉不手术 | 时间、应激表现、生命体征 |
| Anesthesia | 异氟烷麻醉60分钟，无手术 | 诱导/维持/停止时间、呼气末浓度、生命体征、苏醒时间 |
| Surgery | 异氟烷麻醉60分钟 + 标准化腹部小切口/肠管暴露 | 同上 + 切口时间、暴露时间、术中事件、镇痛记录 |

### 7.3 恢复期VR记录

每个时间点（2h、6h、24h、7d）执行同一流程：

1. 兽医确认动物达到清醒、可站立、可安全跑台任务状态。
2. 连接穿入式电极、Neuropixel、视频、IMU、跑台和VR。
3. 记录5分钟静息基线。
4. 按“走”启动-直线迷宫滚动-马赛克奖励区停止取食-继续前进-“停”结束的结构执行VR任务，目标20-40分钟；若动物疲劳可分2个block。
5. 记录5分钟任务后静息。
6. 完成当场质控：同步、噪声、可用trial数、奖励/摄食记录完整性、动物状态。

## 9. 质控与暂停标准

### 9.1 行为质控

| 指标 | 合格标准 |
|---|---|
| 单session有效trial数 | ≥ 20个完整或可判读trial，或达到动物耐受上限并记录原因 |
| 基线完整trial成功率 | > 70% |
| 黑白马赛克奖励区停止成功率 | > 60%，进入正式实验前稳定提升至>70%为优先 |
| 摄食完成记录完整率 | > 95% trial |
| 摄食后继续前进成功率 | > 70% |
| 语音TTL完整率 | 100% |
| 马赛克区和奖励TTL完整率 | 100% |
| 视频可判读率 | > 95% trial |
| 跑台/VR位置日志完整率 | > 95% trial |

### 9.2 电生理质控

| 指标 | 合格标准 |
|---|---|
| Neuropixel可用通道 | > 300个优先；低于阈值需PI判定是否保留 |
| 穿入式电极可用通道 | 每目标区至少50%通道可用 |
| 同步偏差 | TTL跨系统偏差 < 1 ms |
| 噪声 | Neuropixel < 30 μV RMS；穿入式电极 < 50 μV RMS为优先目标 |
| Spike sorting | ISI违反率、隔离度等按预设QC表记录，不达标单元不进入单元级分析 |

### 9.3 动物福利暂停标准

出现以下情况立即暂停实验并通知兽医：

- 持续疼痛评分超出批准SOP阈值。
- 明显感染、出血、神经功能异常、拒食或体重快速下降。
- 跑台任务中出现跌倒、呼吸异常、过度恐惧或持续拒训。
- 头部组件松动、皮肤破损或电极外露风险。

是否退出实验由兽医和PI共同决定，并记录原因。

## 10. 数据分析计划

### 10.1 主要终点

- 完整VR trial成功率。
- 黑白马赛克奖励区停止成功率。
- “走”指令启动潜伏期。
- 奖励区停止潜伏期和停止位置误差。
- 摄食时长及摄食后再启动潜伏期。
- “停”指令结束停止潜伏期。
- 海马位置细胞空间信息量。
- 海马θ-γ PAC。
- 听觉皮层“走/停”语音诱发响应幅度和潜伏期。
- 运动规划皮层启动、奖励区停止、摄食后再启动和最终停止前活动调制。

### 10.2 统计模型

首选混合效应模型：

```text
Outcome ~ Group * Timepoint + (1 | DogID)
```

必要时加入协变量：

```text
Outcome ~ Group * Timepoint + Sex + Weight + BaselinePerformance + (1 | DogID)
```

对于trial级行为数据：

```text
CompleteTrialSuccess ~ Group * Timepoint + TrialIndex + (1 | DogID)
MosaicStopSuccess ~ Group * Timepoint + TrialIndex + (1 | DogID)
```

对连续型行为指标分别建模：

```text
LatencyOrError ~ Group * Timepoint + TrialIndex + (1 | DogID)
```

### 10.3 关键比较

1. Surgery vs Control：术后认知与神经环路损害。
2. Surgery vs Anesthesia：手术创伤叠加效应。
3. Anesthesia vs Control：单纯麻醉效应。
4. T=2h vs baseline、T=24h vs baseline、T=7d vs baseline：恢复时程。

### 10.4 关联分析

- 听觉皮层语音响应 vs “走”启动潜伏期和“停”结束停止潜伏期。
- 运动规划皮层准备活动 vs 启动、奖励区停止、摄食后再启动和最终停止准确性。
- 海马位置细胞空间信息量 vs 黑白马赛克奖励区停止位置误差。
- 海马θ-γ PAC vs 完整VR trial成功率和奖励区停止成功率。
- 跨脑区相干性/Granger信息流 vs 行为恢复速度。

## 12. 每日执行清单

### 实验前

- [ ] 动物健康和疼痛评分通过。
- [ ] 当日DogID、组别、时间点核对。
- [ ] 语音TTL测试通过。
- [ ] 黑白马赛克奖励区出现TTL和食物奖励装置测试通过。
- [ ] 跑台、VR、IMU、视频测试通过。
- [ ] 穿入式电极和Neuropixel阻抗/噪声检查完成。
- [ ] CRF表格和文件夹创建完成。

### 实验中

- [ ] 每个block记录开始/结束时间。
- [ ] 标记“走”、马赛克奖励区出现、食物奖励投放、奖励获取、摄食完成、继续前进和“停”的关键事件。
- [ ] 标记所有中断、异常trial和动物状态变化。
- [ ] 若动物疲劳或拒训，暂停并记录。
- [ ] 若信号异常，先记录时间再排查，不覆盖原始数据。

### 实验后

- [ ] 动物交还兽医/饲养人员并完成福利记录。
- [ ] 导出VRTrials、视频、神经数据和CRF。
- [ ] 完成双备份和校验。
- [ ] 填写SignalQC和当天实验小结。
- [ ] 团队10分钟复盘，决定是否调整下一次session。
